# Lens Filter Tray

This is a box for holding lens filters of a variety of sizes.
I started with a design using [Bento3D](https://bento3d.design/tray)
and modified it with cutouts to pull out the filters and a curved bottom
so that filters don't roll around.

The Bento3D tray is 160x160x65mm, stackable. The smaller row is 65mm wide 
(inside). This is big enough to hold up to a standard 58mm lens
filter. Larger filters can be placed in the center row or
using the 2nd tray with the larger row. Larger filters will get in the way
of stacking the trays or adding a lid. Of course, a taller version of the tray
could be generated with Bento3D and modified.

I've included a lid design, though one can also be generated with Bento3D,
as a 160x160x5mm tray with no dividers.
